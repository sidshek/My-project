About This Project

Hey, I'm Siddharth Shekhar. I built this website because I was bored, but also wanted to explore how websites work using HTML, CSS, and JavaScript.

It started as a simple project but turned into a fun way to teach beginners how to build stuff on the web. This site uses Tailwind CSS for quick styling and Bootstrap for UI components.
